# Cyber Crime Reporting System - User Guide

## Getting Started

### Accessing the System
1. Open your web browser
2. Navigate to the Cyber Crime Reporting System URL
3. No registration required for anonymous reporting

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- For evidence upload: Files up to 1GB for videos, 50MB for images

## Reporting a Cybercrime

### Step 1: Choose Reporting Method
- **Anonymous**: No personal information required
- **Registered**: Provide contact details for follow-up

### Step 2: Fill Incident Details
- **Date of Incident**: When the crime occurred
- **Location**: Where the incident took place
- **Type of Cybercrime**: Select from predefined categories
- **Description**: Detailed account of what happened

### Step 3: Upload Evidence (Optional)
- **Supported Formats**:
  - Videos: MP4, MOV, AVI (max 1GB)
  - Images: JPG, PNG (max 50MB each)
  - Documents: PDF (max 10MB)
- **Security**: All files are encrypted and scanned for malware

### Step 4: Submit Complaint
- Review all information
- Click "Submit Complaint"
- Save the tracking ID for future reference

## Tracking Your Complaint

### Using Tracking ID
1. Go to the main page
2. Use the tracking feature (when implemented)
3. Enter your tracking ID
4. View complaint status and updates

### What Happens Next
1. **Submission**: Complaint recorded in secure database
2. **AI Processing**: Automatic categorization and summarization
3. **Review**: Authorities review the complaint
4. **Investigation**: Case assigned to investigators
5. **Updates**: Status updates provided via tracking ID

## Evidence Preservation

### Before Reporting
- **Don't delete** any suspicious emails, messages, or files
- **Take screenshots** of relevant screens
- **Save URLs** and web addresses
- **Record dates and times** of incidents
- **Document financial losses** with transaction records

### During Investigation
- Cooperate with authorities
- Provide additional evidence when requested
- Keep all original evidence intact

## Frequently Asked Questions

### Reporting
**Q: Can I report anonymously?**
A: Yes, anonymous reporting is available and encouraged.

**Q: What if I don't have evidence?**
A: Evidence is helpful but not required. Detailed descriptions are valuable.

**Q: How long does the process take?**
A: Initial submission is immediate. Investigation time varies by case complexity.

### Technical Issues
**Q: File upload failed**
A: Check file size limits and supported formats. Try smaller files or different formats.

**Q: System is slow**
A: Large file uploads may take time due to security scanning. Please be patient.

### Legal Questions
**Q: What laws apply to my case?**
A: Use the Cyber Law Guide to find applicable PECA sections.

**Q: Can I withdraw my complaint?**
A: Contact authorities with your tracking ID to discuss modifications.

## Best Practices

### Secure Reporting
- Use a private device if possible
- Avoid public Wi-Fi for sensitive reports
- Save tracking ID securely
- Use strong passwords if registering

### Evidence Handling
- Create backups of evidence
- Don't modify original files
- Document chain of custody
- Report as soon as possible

### Follow-up
- Check status regularly
- Respond to authority requests promptly
- Keep records of all communications
- Seek support if needed

## Troubleshooting

### Common Issues
1. **Form not submitting**: Check required fields and data formats
2. **File upload errors**: Verify file type and size limits
3. **Tracking ID not working**: Ensure correct format (CCRS-PK-YYYY-XXXXXX)

### Getting Help
- Check FAQ section
- Contact support
- Use emergency contacts for urgent issues

## Legal Notice

This system is for reporting cybercrimes under Pakistan's Prevention of Electronic Crimes Act (PECA) 2016. All reports are taken seriously and investigated according to applicable laws and procedures.

**Emergency**: If you are in immediate danger, call 15 immediately.
