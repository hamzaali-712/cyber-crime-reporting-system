# 🛡️ Cyber Crime Reporting & Case Management System

A **Python-based Cyber Crime Reporting & Case Management System** developed as a **Software Engineering group project**.  
This project replicates the **NCCIA Pakistan Cyber Crime Reporting Portal**, integrating real-world workflows for **electronic evidence management** and case tracking.

---

## 📌 Overview

- Built using **Python (Django Framework)** with **MySQL database integration**.  
- Designed to allow **citizens to report cyber crimes online**, upload evidence, and track case status.  
- Provides administrators with tools to **verify complaints, assign cases, update status, and generate reports**.  
- Inspired by the **National Cyber Crime Investigation Agency (NCCIA) Pakistan portal**, aiming to simulate real-world functionality.  
- Integrated **live electronic evidence handling** (screenshots, PDFs, images, links) for authenticity and practical application.  

---

## ⚙️ Features

### 👤 User Side
- Account creation & login system.  
- Submit cyber crime complaints with evidence uploads.  
- Real-time case tracking (Pending / Investigating / Closed).  
- Notifications for case updates.  

### 🛠️ Admin Side
- Complaint verification & approval/rejection.  
- Case assignment to officers.  
- Status updates & progress tracking.  
- Report generation (monthly, type-based, graphical).  

### 📊 Case Management
- Unique Case IDs for each complaint.  
- Officer assignment & workload tracking.  
- Evidence stored securely in database.
